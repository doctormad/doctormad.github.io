Open-Source apps for makerspaces

Depending on the specific field, problem to solve, and application, there are many free and open-Source software (FOSS) tools that could be useful in STEM education, for students, for teachers and makerspaces.

Free and Open-Source software (FOSS) is software that gives users the freedom to use, study, share, and modify the code. FOSS tools are backed by global communities of enthusiasts who write documentation, create tutorials, and help beginners solve problems.

For a makerspace, FOSS is more than just zero-cost software tools; it adheres to and follows the philosophy of "hacking," sharing, and community that defines the maker movement. FOSS effectively removes the "paywall" to education.

FOSS is a powerful iteration of the key principles of the "Philosophy of sharing" from Mark Hatch’s 2013 "The Maker Movement Manifesto", a cornerstone of how modern Makerspaces operate: "Make, Share, Give, Learn, Tool Up, Play, Participate, Support, and Change.".

Key STEM-advantages

* Zero-Cost Licensing: free to deploy across entire classrooms or labs, no licensing fees, activation keys, subscriptions or hidden costs. There is no "dependancy" on commercial actors or models.

* Source code:underlying code is freely readable, available and modifiable. It also empowers the community to modify or repair the software, preventing it from becoming obsolete if the original developer stops supporting it.

* Skill development: by using FOSS you're thinking out-of-the-box already; by learning and adapting to (new) "software principles" instead of learning "How to do X in [Brand Name]," knowledge becomes transferable and "*mutable*".

Practical example: mastering image editing & design using GIMP or Inkscape transfers those skills to other software such as Photoshop or Affinity. Knowing only Photoshop's shortcuts can make you feel "lost" or "stuck" without them.


STEM-applications & FOSS apps

- Note: The followinging list of FOSS applications is intended for use on the Linux, Microsoft Windows and Mac OS operating systems.

For Linux systems most of these apps can be found in their respective software centers or using simple install commands in the terminal. For proprietary OS systems such as Microsoft Windows and Mac OS install files need to be downloaded from the provided FOSS urls.

Basic installation instructions

  • For Linux systems most of these apps can be found in their respective software centers, software managers or using basic install commands in the terminal.

  • For proprietary OS systems such as Microsoft Windows and Mac OS install files need to be downloaded from the provided FOSS websites and download options.
